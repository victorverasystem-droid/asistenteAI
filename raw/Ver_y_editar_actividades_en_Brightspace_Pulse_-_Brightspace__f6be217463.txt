TYPE: web

Brightspace Pulse muestra las actividades asignadas a partir de los datos correspondientes en Brightspace. Estas actividades pueden incluir asignaciones (asignadas a partir de una asignación, un tema/foro de debate, una lista de control o encuesta), evaluaciones (asignadas a partir de un cuestionario o elemento de calificación) y lecturas (asignadas a partir de un tema de contenido).
